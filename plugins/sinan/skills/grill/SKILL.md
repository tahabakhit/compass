# Grill

Use this skill when the next useful move is to reduce ambiguity before planning or implementation.

## Workflow

1. Identify the decision that blocks progress.
2. Ask one to three short questions at a time.
3. Prefer concrete tradeoffs over broad brainstorming.
4. Stop grilling once acceptance criteria, constraints, and the next action are clear.
5. If the request becomes implementation work, hand off to the `implement` workflow.

## Output

End with the clarified decision, the accepted constraints, and the smallest next step.
