# Handoff

Use this skill when work needs to be resumed later or transferred to another agent.

## Include

- Current objective and repo path.
- Files changed or most relevant files read.
- Decisions made and why.
- Validation commands and results.
- Known blockers, risks, or unfinished tasks.
- The next concrete step.

Keep it compact enough to be useful as startup context.
